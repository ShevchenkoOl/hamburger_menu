Preprosessor SASS

SASS is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS).

It is an extension of CSS that enables you to use variables, nested rules, mixins, functions, and more, all within a CSS-compatible syntax.

It helps to keep your stylesheets organized and makes it easier to maintain and scale your CSS code.
